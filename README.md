# A Group of Layout Pattern Generators.

## Dependencies

- klayout 0.25.6

- python 3.6.x

- numpy

- pandas

## Generators

```mkdir -p layouts```

- Uni-direction Metal Layer

```make m2_test.oas```

- Via Layer

```make via_test.oas```
